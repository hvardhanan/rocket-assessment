# UC002 - Library Management System
Develop a simple Library Management System

## Logic and Approach used

Created classes such as User, Librarian, Member, BorrowRecord and FinePolicy. 

We use an in memory storage for the records and books inside the Librarian object instantiated in main.

We instantiate a librarian and member object and then use it to do operations. We don't use any auth in this case as it added slight complexity to the question which was asked.

## Execution Steps
Running the main file using python is enough to run this code. 

The code was tested using version 3.14.0 in the local dev environment.

No external dependencies are required to run this project.

```
python main.py
```
